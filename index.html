API Key Settings

Anthropic API key

When a key is saved, the app uses Claude to analyse the actual photographs and generate an examiner prompt, model speech, question and answer that describe what is genuinely visible in your images.

Your key is stored only in your browser's localStorage — it is never sent to GitHub or any server other than Anthropic's API directly. Without a key the app falls back to topic-based model texts.

OpenAI API key (for TTS voices)

When set, the app uses OpenAI Text-to-Speech with two distinct voices: onyx (deep male) for the examiner and nova (female) for the candidate speech.

Your key is stored only in your browser's localStorage — it is never sent to GitHub or any server other than OpenAI's API directly. Without a key the app falls back to your browser's built-in speech synthesis.
